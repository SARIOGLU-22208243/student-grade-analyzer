# Student Grade Analyzer

This project is a simple Python application that analyzes student grades.

## Features
- Add student names and grades
- Display all students
- Calculate average grade
- Show highest and lowest grades
- Count passed and failed students

## Technologies Used
- Python

## Purpose
This project was created to practice Python fundamentals such as:
- Lists
- Dictionaries
- Loops
- Conditions
- Functions
- Basic data analysis
